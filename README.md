# Loonshot Exploration

Personal workspace for exploring ambitious ideas using the Loonshot Framework - 20 minutes a day.

## Structure

- `framework/` - Reference documentation for the Loonshot Framework methodology
- `explorations/` - Individual idea explorations (one folder per idea)
- `explorations/templates/` - Templates for weekly workflow documents

## Getting Started

1. Read [framework/loonshot-framework.md](framework/loonshot-framework.md) for the full methodology
2. Use templates in `explorations/templates/` to start a new exploration
3. Create a new folder under `explorations/` for each idea you pursue

## Quick Reference

| Day | Activity |
|-----|----------|
| Friday | Generate 10 ideas, narrow with AI |
| Monday | Select idea, frame with 4 sentences |
| Tuesday | Identify barriers & constraints |
| Wednesday | Decompose & define success metrics |
| Thursday | Build AI agents & tools |
| Friday | Run agents, iterate, journal learnings |
